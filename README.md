# POWER-DIFFERENCE-ANALYSIS-BETWEEN-REVERSIBLE-AND-NORMAL-LOGIC-GATES-USING-VIVADO
Verilog implementation and Vivado simulation of Conventional and Reversible Carry Select Adders (CSA). The project compares power, delay, and utilization to analyze low-power design using reversible logic gates such as Peres and Feynman, following the methodology from the ICAMMSD-2024 research paper.
